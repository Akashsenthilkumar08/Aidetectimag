About AI Detect

AI Detect is a deep learning–based image authentication platform developed as a major academic project. It enables users to identify whether an uploaded image is AI-generated or real using an advanced CNN model with Grad-CAM visualization. The system also provides user profiles, detection history, analytics, AI news, and secure cloud storage using Firebase.
